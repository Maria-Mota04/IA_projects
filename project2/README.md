## How to Run the Project

### 1. Go to the project root directory

You should run everything from the root of the project:

```bash
cd project2
```

---

### 2. Install dependencies

Make sure the `requirements.txt` file is in the project root.

```bash
pip install -r requirements.txt
```

---

### 3. Run the Flask application

Navigate to the web application folder:

```bash
cd webapp
```

Then run:

```bash
python3 app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000
```
